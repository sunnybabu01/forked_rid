HTML Anchor Tag

➢The HTML anchor tag (<a>) defines a hyperlink that links one page to another.It can create hyperlink to other web page as well as files, location, or any URL. The "href" attribute is the most important attribute of the HTML a tag. and which links to destination page or URL.

Where:

➢ href attribute is used to define the address of the file to be linked.

Syntax

<a href="URL">Link Text</a>

Example:

<a href="thirdpage.html">Click for third Page</a>

Note:

• The target attribute can only use with href attribute in anchor tag.
• If we will not use target attribute then link will open in same page.

Using the Target Attribute

The target attribute specifies where to open the linked document:

  • target="_blank": Opens in a new window/tab.
  • target="_self": Opens in the same frame.
  • target="_parent": Opens in the parent frame.
  • target="_top": Opens in the full body of the window.

Examples

  • <a href="https://www.twksaa.org/" target="_blank">this-link</a>
  • <a href="https://www.twksaa.org/" target="_self">this-link</a>
  • <a href="https://www.twksaa.org/" target="_parent">this-link</a>
  • <a href="https://www.twksaa.org/" target="_top">this-link</a>
  • <a href="https://www.twksaa.org/">this-link</a>

❖Appearance of HTML Bookmarks:

  • An unvisited link is displayed underlined and blue.
  • A visited link is underlined and purple.
  • An active link is underlined and red.

❖ HTML Links-Creat Bookmarks:

➢ HTML links can be used to create bookmarks, so that readers can jump to specific parts of a web page.

➢ Bookmarks can be useful if a web page is very long.

➢ To create a bookmark - first create the bookmark, then add a link to it.

➢ When the link is clicked, the page will scroll down or up to the location with the bookmark.
