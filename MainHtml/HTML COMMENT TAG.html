HTML COMMENT TAG

➢You can add comments to your HTML source by using the following syntax:

➢<!-- Write your comments here -->

➢Notice that there is an exclamation point (!) in the start tag, but not in the end tag.

➢Comments can be used to hide content.

Note: Comments are not displayed by the browser.

Example:

Example Hide a section of HTML code:
<!-- This is a comment --> <p>This is a paragraph. </p>
<p>This is a paragraph. </p> <!--
<!-- Remember to add more information here --> <p>Look at this cool image:</p>

Example Hide a section of HTML code:

 <img border="100" src="t3.jpg" alt="image">
<p>This is a paragraph. </p> -->
<p>This is a paragraph. </p>
